# GOAT Battle Arena

🔗 **Live Demo:** [https://football-legends-roast-battle.netlify.app/](https://football-legends-roast-battle.netlify.app/)

## Overview
GOAT Battle Arena is an AI-powered football debate simulator that transforms legendary football rivalries into intense, round-by-round battles. Instead of comparing players through simple statistics, the application creates dynamic debates where football icons challenge each other's achievements, failures, controversies, and defining career moments.

From the endless Messi vs Ronaldo debate to classic comparisons like Maradona vs Pelé, GOAT Battle Arena uses AI to generate arguments based on real football history and presents them in an entertaining battle format. Each round highlights different aspects of a player's legacy, allowing users to explore football's greatest debates from multiple perspectives.

The project is designed for football fans who enjoy discussing the greatest players of all time and want a fun, interactive way to compare legends.

## Features
* Select any two football legends for a head-to-head battle
* AI-generated arguments, counterarguments, and roast-style responses
* Uses real career achievements, trophies, records, setbacks, and controversies
* Multiple battle rounds covering different aspects of a player's career
* AI-powered winner analysis and final verdict
* Dynamic player images and battle presentation
* Responsive design for desktop, tablet, and mobile devices
* Fast and lightweight front-end implementation

## How It Works
1. Choose two football legends from the available list.
2. Enter your Groq API key.
3. Start the battle.
4. The AI analyzes both players using publicly known football history.
5. Each round focuses on a specific category such as trophies, individual awards, international success, longevity, influence, or controversies.
6. The players exchange AI-generated arguments and responses.
7. After all rounds are completed, the AI evaluates the debate and declares a winner.

## Setup
This project uses the Groq API to generate battle arguments and final verdicts.
1. Create a Groq account.
2. Generate an API key from the given link:
   https://console.groq.com/keys
3. Copy your API key.
4. Paste the key into the application's API key field.
5. Start generating football battles.

**Important:** Never expose your API key publicly or commit it to your GitHub repository.
